    What we Do

    This Web Application Comes with an integrated Naive Bias Model

    Which is a Machine Learning Model Mainly used for Predicting whether your Dream Car really matches your standards or not. Basically the Machine Learning Model is fired using a Django Backend which is completely based in Python.

    • To predict we need to select the symptoms among them and click on button predict.
    • As soon as the predict button is hit, Django backend activates and the Supervised model which is already trained using explicit dataset accepts the values.
    • Then converts it to model understandable Language (Data Preprocessing) and predicts the outcome using the predict operation.
    • This predicted result is then flushed to the frontend of the website for user notice.

    Note:- The result is totally based on prediction. We don't register any claim on the truth of this information.
