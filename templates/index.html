    What we Do

    This Web Application Comes with an Integrated Naive Bayes Model

    It is a Machine Learning Model Mainly used for Predicting whether your Dream Car really matches your standards or not. Basically the Machine Learning Model is fired using a Django Backend which is completely based in Python.

    • For Prediction, we need to provide few details about the Vehicle
    • Then using this Data, the model fires up gets activated to Predict the Outcome based on the given Details.
    • The Data is then Converted to a form in which the Database has its keys.
    • The Prediction Result is then converted to User understandable form and displayed on the Web Page for viewing purpose.

    Note:- The result is totally based on prediction. We don't register any claim on the truth of this information.
